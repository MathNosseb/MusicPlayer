# Music Scraper
(Vibe Coded 💻)
Un script Python pour récupérer automatiquement toutes les musiques de PLK, y compris albums, singles et featurings, et les stocker localement en JSON.

---

## 📦 Contenu du projet

- `scraper.py` – Script principal pour récupérer les musiques depuis les API ou sites web.
- `data/` – Dossier où sont sauvegardées les données JSON.
- `requirements.txt` – Bibliothèques Python nécessaires.

---

## 🚀 Fonctionnalités

- Récupère **tous les albums officiels** de PLK.
- Récupère les **singles** et **featurings**.
- Sauvegarde les données dans un fichier JSON structuré.
- Gère les doublons et trie les morceaux par date.
